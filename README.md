# FGDA
Code for paper "Gradient Distribution Alignment Certificates Better Adversarial Domain Adaptation" which has been accept by ICCV2021.

## Requirements
- Python 3.5
- torch 1.1.0
- torchvision 0.3.0

## Training
```
train_FGDA_CDANE.py
```
```
train_FGDA_MDD.py
```
